# ✈️ AI Travel Planner

An AI-powered travel planner that generates day-wise itineraries based on user input like destination, budget, and interests.

---

## 🚀 Features

- Chat-based travel planning interface  
- Generates structured day-wise itineraries  
- Accepts natural language input  
- Place validation to remove incorrect or irrelevant locations  
- Clean and readable output format  
- Runs locally using Ollama (no external APIs required)  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Ollama (Local LLM)  
- Prompt Engineering
